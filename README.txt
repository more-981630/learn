minishell：
	minishell 是一个命令处理器，处理系统外部命令和cd、exit命令。
	外部命令格式：简单命令 [< 文件名] [ | 简单命令] [>/>> 文件名] [&]
		     （其中[]表示可选项，/表示前后仅可选择一个）
	cd命令：cd 相对路径 
	exit命令：exit
	具体介绍请转/learn/minishell/README.txt
sudoku：
	sudoku是一个9*9的数独游戏，游戏规则是将1-9放入每一行、每一列、每一个宫中，且要满足不能出现重复的数字。
	sudoku具有交互界面，用户可以选择开始游戏、填入数字、清楚数字、重置游戏、答案、退出等操作。
	具体介绍请转/learn/sudoku/README.txt
